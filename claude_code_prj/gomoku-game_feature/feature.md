# 五子棋项目新功能需求

## 1. 对局模式

- 支持人机对战、双人本地对战和观战复盘三种模式。
- 人机对战允许选择 AI 难度：入门、普通、高手。
- 支持先手方选择，玩家可以选择黑棋或白棋。

## 2. 棋局体验

- 增加悔棋功能，允许玩家回退最近一步或一轮落子。
- 增加重新开始和保存当前棋局功能。
- 增加落子提示，高亮最近一步棋和推荐落点。
- 增加禁手规则开关，满足不同五子棋规则玩法。

## 3. 计时与胜负

- 支持倒计时模式和不限时模式。
- 倒计时结束后自动判负，并展示明确的结果提示。
- 胜负判定需要展示获胜连线，方便玩家确认结果。
- 支持平局判定，当棋盘下满且无人获胜时结束对局。

## 4. 复盘与记录

- 保存每局棋谱，包括落子顺序、对局时间、胜负结果和双方信息。
- 提供棋谱回放功能，支持上一步、下一步、自动播放和跳转到指定步数。
- 支持导出棋谱为 JSON 文件，方便后续分析或分享。

## 5. 界面与交互

- 适配桌面端和移动端屏幕，棋盘在不同设备上保持清晰可操作。
- 增加主题切换，至少支持浅色、深色和木纹棋盘主题。
- 添加音效开关，包括落子音、胜利提示音和倒计时提醒音。
- 对按钮、弹窗和状态提示统一视觉风格，提高整体一致性。

## 6. 可维护性

- 将棋盘渲染、规则判断、AI 逻辑和状态管理拆分为清晰模块。
- 为胜负判定、禁手判断和 AI 决策增加单元测试。
- 关键逻辑添加中文注释，便于后续维护和协作。
