# XAC模板生成器
根据用户指定的应用、环境、组件及参数生成完整的XAC模板文件

## 生成规则
- 1.先生成一个空模板，文件目录结构参考‘file_pattern.md’来创建
- 2.只生成用户指定的组件，需要核对用户指定的组件是否在‘component-types.md’中，不在其中的组件直接跳过，并继续生成剩余支持的组件
- 3.组件参数需读取‘component-example.md’来命名并生成
- 4.组件实参需要读取‘component-config.md’来匹配生成
- 5.根据用户指定的组件执行顺序，读取‘component-workflow.md’来生成执行工作流文件
- 6.以上生成的组件数据需要立即写入到本地

## 输入处理
输入应用ID、组件名称、spec环境等信息


## 输出格式
输出固定格式的模板文件


## 质量检查

文件生成后，快速检查：
- 目录结构是否正确
- 文件是否齐全
- 配置文件中组件是否齐全
- 配置文件中组件参数是否齐全
- 参数命名是否正确，参数是否正确
