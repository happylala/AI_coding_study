# 五子棋游戏

一个使用原生 JavaScript、HTML 和 CSS 开发的前端五子棋游戏。

## 项目结构

```
gomoku_game/
├── index.html      # 主HTML文件
├── style.css       # 样式文件
├── game.js         # 游戏主控制逻辑
├── board.js        # 棋盘渲染和UI操作
├── gameLogic.js    # 游戏规则和状态管理
└── README.md       # 项目说明文档
```

## 功能特性

- 15x15 标准五子棋棋盘
- 双人对战模式（黑白轮流下棋）
- 自动判断胜负（五子连珠）
- 支持重新开始游戏
- 响应式设计，支持移动端
- 美观的UI界面和动画效果

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件即可开始游戏
2. 或者使用本地服务器：
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 然后在浏览器访问
   # http://localhost:8000
   ```

## 游戏规则

1. 黑方先手，白方后手
2. 玩家轮流在棋盘上落子
3. 先在横、竖、斜任意方向形成五子连珠者获胜
4. 棋盘下满且无人获胜则为平局

## 技术栈

- HTML5
- CSS3（使用Grid布局）
- 原生 JavaScript（ES6+ 模块化）

## 代码规范

- 使用 ES 模块（import/export）语法
- 使用解构导入
- 代码添加中文注释
- 遵循面向对象编程原则

## 项目亮点

- 模块化设计：将UI渲染、游戏逻辑、主控制分离
- 清晰的代码结构和详细的中文注释
- 优雅的视觉效果和用户体验
- 无需任何外部依赖库
